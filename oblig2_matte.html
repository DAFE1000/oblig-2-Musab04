Funksjon: f(x) = e−x/4 · arctan(x)

Del 1 – Analytiske utrekningar

For å finna maksimalpunktet deriverer me funksjonen og set f'(x) = 0.

Me bruker produktregelen: (u · v)' = u'v + uv'

La u = e−x/4 og v = arctan(x)

    u' = −1/4 · e−x/4
    v' = 1 / (x² + 1)

Produktregelen gjev:

    f'(x) = −1/4 · e−x/4 · arctan(x) + e−x/4 · 1/(x²+1)

Faktoriser ut e−x/4:

    f'(x) = e−x/4 · [ 1/(x²+1) − arctan(x)/4 ]

Sett f'(x) = 0. Sidan e−x/4 > 0 for alle x, må den andre faktoren vera null:

    1/(x²+1) − arctan(x)/4 = 0

Multipliser med 4:

    4/(x²+1) − arctan(x) = 0

∴  arctan(x) − 4/(x²+1) = 0    (QED)

Sidan e−x/4 alltid er positiv, er det forteiknet til [ 1/(x²+1) − arctan(x)/4 ] åleine som avgjer om f er stigande eller avtagande. Dette viser at det finst nettopp eitt kritisk punkt.

Del 2 – Numeriske utrekningar (Python)

import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import brentq

# Funksjonen og den kritiske likninga
def f(x):
    return np.exp(-x/4) * np.arctan(x)

def g(x):
    # g(x) = arctan(x) - 4/(x^2+1)  ->  loys g(x) = 0
    return np.arctan(x) - 4 / (x**2 + 1)

# Finn rota numerisk med Brent-metoden
x_max = brentq(g, 0.1, 5.0)
y_max = f(x_max)

print(f"x-koordinat: {x_max:.6f}")
print(f"y-koordinat: {y_max:.6f}")
print(f"Kontroll g(x_max) = {g(x_max):.2e}")

# Plott
x = np.linspace(-3, 10, 800)
y = f(x)

fig, ax = plt.subplots(figsize=(9, 5))
ax.plot(x, y, label=r'$f(x)=e^{-x/4}\cdot\arctan x$')
ax.axhline(0, color='gray', linewidth=0.8)
ax.axvline(0, color='gray', linewidth=0.8)
ax.scatter([x_max], [y_max], color='red', s=80, zorder=5, label='Toppunkt')
ax.annotate(
    f"Toppunkt\n({x_max:.4f}, {y_max:.4f})",
    xy=(x_max, y_max), xytext=(x_max + 1.2, y_max - 0.08),
    arrowprops=dict(arrowstyle='->')
)
ax.set_title(r'$f(x)=e^{-x/4}\cdot\arctan x$')
ax.set_xlabel('x')
ax.set_ylabel('f(x)')
ax.legend()
ax.grid(True, alpha=0.3)
plt.tight_layout()
plt.savefig('toppunkt_plot.png', dpi=150)
plt.show()
